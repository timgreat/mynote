## R输入输出

用`print()`函数显示某个变量或表达式的值，用`cat()`函数把字符串、变量、表达式连接起来显示， 其中变量和表达式的类型一般是标量或向量，不能是矩阵、列表等复杂数据。**注意`cat()`显示中需要换行需要在自变量中包含字符串`"\n"`**。

`cat()`默认显示在命令行窗口， 为了写入指定文件中， 在`cat()`调用中用`file=`选项， 这时如果已有文件会把原有内容覆盖， 为了在已有文件时不覆盖原有内容而是在末尾添加， 在`cat()`中使用`append=TRUE`选项。

```R
cat("=== 结果文件 ===\n", file="res.txt")
cat("x =", x, "\n", file="res.txt", append=TRUE)
```

函数`sink()`可以用来把命令行窗口显示的运行结果转向保存到指定的文本文件中， 如果希望保存到文件的同时也在命令行窗口显示， 使用`split=TRUE`选项，最后使用`sink()`结束记录。

```R
sink("allres.txt", split=TRUE)
...
sink()
```

 用`save()`命令要求把指定的若干个变量（直接用名字，不需要表示成字符串） 保存到用`file=`指定的文件中， 随后可以用`load()`命令恢复到工作空间中。

```R
save(scores, file="scores.RData")
load("scores.RData")
#save multi variants
save(x, zeta, file="vars.RData")
save(list = c("x", "zeta"), file="vars.RData")
```

对于一个数据框， 可以用`write.csv()`  保存文件：

```R
da <- tibble("name"=c("李明", "刘颖", "张浩"),
    "age"=c(15, 17, 16))
write_csv(da, path="data.csv")
```
